# Whale Tracker Skill

## Purpose
Track Bitcoin accumulators (MSTR, GIGA, ETC, etc.) and generate educational/trading signals content for X (Twitter).

## Data Sources
- Blockchair API (free tier: 5K calls/month)
- CoinGecko API (free)
- Finnhub (for stock data)

## Configuration
Add to TOOLS.md:
```
### Whale Tracker
- Blockchair API Key: (get from blockchair.com)
- Twitter API Keys: (get from developer.x.com)
```

## Tracking Targets
- MSTR (MicroStrategy)
- GIGA (Giganet)
- ETC (Electrum)
- Other public BTC holders

## Signals to Track
1. Large transactions (>100 BTC)
2. Exchange inflows/outflows
3. Accumulator announcements
4. On-chain metrics (HODL waves, SOPR, MVRV)

## Posting Strategy
- Educational: Explain what whale movements mean
- Signals: "Large wallet moved X BTC - monitoring"
- Analysis: "Based on on-chain data, potential accumulation pattern"

## Disclaimer Required
"Educational only. Not financial advice. Do your own research."
